## 具体操作

### 立即检查

1. **TK App → 设置 → 账号状态（Account Status）** — 看有没有黄色/红色标记
2. **TK App → 收件箱（Inbox）** — 看有没有"video under review""didn't pass review"通知
3. **已发布的视频 → 点进去 → 右上角"..."→ Analytics** — 看状态是"under review"还是"live"

### 持续操作

| 动作 | 说明 |
|:--|:--|
| **每天发1-2条** | 不批量、不堆量。新号限发阶段 |
| **纯原创内容** | 不搬运、不混剪 |
| **检查hashtag** | 不用#fyp #viral等被滥用标签，用#asmr #naturalmaterials #seedpod |
| **检查文字** | 不用"fake""real vs fake"，改"Real or not?" |
| **等** | 新号沙盒24-72小时，最多7天。预计6/20前后恢复正常 |

### 什么时候需要放弃

- 发了20+条视频，全部0播放
- 账号状态出现红色违规标记
- 超过14天仍未恢复

## 关于Shadowban的排查

如果沙盒期过后仍0播放，按以下排查：

| 排查项 | 检查 |
|:--|:--|
| 网络/IP | 是否使用稳定的美国IP？节点是否被标记？ |
| 设备 | 手机语言/地区是否设为美国？是否插了国内SIM卡？ |
| 内容 | 是否有搬运/重复内容？BGM是否有版权？ |
| 标签 | 是否用了被限的hashtag？OCR扫描文字是否违规？ |
| 行为 | 是否短时间大量关注/点赞/评论？ |
