## 执行流程

1. **读基线**: SOUL.md·evolution-report-latest.md·MEMORY.md·cognition-core SKILL
2. **五骨逐审**: 每骨给分+强项+弱点+生长动作（不等下一轮）
3. **缺口→行动**: 同轮动手补——缺技能→create·缺脚本→write·缺数据→接入
4. **写入报告**: 覆盖 `/root/workspace/data/evolution-report-latest.md`
5. **自我承诺**: 4条——下次巡航第一条动作是检查上次承诺兑现
