# 习惯体系架构 · 记分卡 vs 追踪

> 基于《原子习惯》James Clear | 2026-06-10 拆分

## 核心区分

| | 习惯记分卡 | 习惯追踪 |
|:--|:--|:--|
| **是什么** | 全量行为审计 | 核心习惯每日打卡 |
| **目的** | 意识——看清楚自己在做什么 | 执行——让对的事每天都做 |
| **范围** | 大而全（上午/下午/晚上所有行为） | 小而精（5项核心习惯） |
| **频率** | 不定期审视 | 每天看·每天打✅ |
| **输出** | 标记倾向（+/-/=） | 连续天数·视觉累积 |
| **对应原著** | 第一步·让习惯显而易见 | 第四步·让它令人满足 |

## 拆分原因

用户纠正：习惯记分卡和习惯追踪是两个东西。之前混在一起是错误的。

## 文件结构

- `习惯记分卡.md` — 全量行为审计·四大定律·身份锚
- `习惯追踪.md` — 5项核心习惯·每日打卡·连续天数
