# A股交易规则速查

> 2026-06-10 | 工业富联判断时暴露的知识缺口

---

## 核心规则

| 规则 | 内容 |
|:--|:--|
| 交易单位 | **1手=100股**。买入必须整手，不能买零股 |
| 价格单位 | 东方财富API返回的价格是**分**（7115=¥71.15） |
| 一手金额 | 股价×100。如¥71.15/股→一手¥7,115 |
| 交易时间 | 9:30-11:30, 13:00-15:00（北京时间） |
| 涨跌幅 | 主板±10%，创业板±20%，科创板±20% |

---

## 东方财富API

```
https://push2.eastmoney.com/api/qt/stock/get?secid=1.601138&fields=f43,f44,f45,f46,f47,f48,f50,f57,f58,f60,f169,f170
```

| 字段 | 含义 | 单位 |
|:--|:--|:--|
| f43 | 现价 | 分 |
| f60 | 昨收 | 分 |
| f44 | 最高 | 分 |
| f45 | 最低 | 分 |
| f46 | 开盘 | 分 |
| f169 | 涨跌额 | 分 |
| f170 | 涨跌幅 | 基点（-480=-4.80%） |
| f47 | 成交量 | 手 |
| f48 | 成交额 | 元 |

---

## 报价惯例

- 必须同时报**单价**和**一手金额**
- 不说"工业富联现价¥7,115"——那是分，不是元
- 正确："工业富联¥71.15/股，一手¥7,115"
- 判断买卖时先算一手金额占现金比例
