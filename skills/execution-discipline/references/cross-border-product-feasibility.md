# 跨境选品可行性框架 · 六步法

> 2026-06-10 | 豆荚禁运→换品实战提炼
> 适用：TK美区FBT自运营·轻小件·手工/天然系

## 六步框架

### 1. 市场信号验证
- TK视频互动率>3%即需求成立
- 豆荚验证：7条视频·平均5.2%互动率·最高8.8%
- 不靠经验≠不靠数据。有数据用数据，没数据拍视频测

### 2. TK平台合规
- 查TK Prohibited Products Policy
- 植物制品禁的是「受保护物种(Lacey Act)+非法+不安全」。普通天然材料不在禁列
- 可能需要category approval——走标准流程

### 3. 物流可行性
- **第一道：云途** — 常见禁运类：植物制品/动物制品/液体/粉末/纯电池/品牌Logo
- **第二道：替代物流** — DHL/燕文/递四方。每个都有不同的禁运清单
- **关键区分：** 物流禁运 ≠ TK平台禁止。云途说不能发≠产品不能卖
- **天然材料红线：** 未加工/可能含虫害的植物制品→几乎所有物流都拒。加工过（上漆/打磨/钻孔）的→DHL可能接受。但一刀切时——换品

### 4. 成本结构
- 采购+物流+FBT履约($3.58/单) = 到仓成本
- 售价≥到仓成本×2 即成立
- 轻小件(<100g)物流成本是关键变量

### 5. 竞争格局
- 物流门槛=竞争壁垒。禁运品=能发的人少=独占赛道
- 但禁运到所有渠道都堵=必须换品

### 6. 内容兼容性
- 必须兼容已验证的视频模板
- 豆荚模板#3：10秒·满足感展示·选择题结尾·8.8%互动率
- 新选的品→能用同模板拍=迁移成本为零

## 换品决策树

```
物流全堵？
  ├─ 是 → 立刻换品（不等不纠结）
  │      ├─ 人工材料替代（树脂仿天然→保留审美方向）
  │      ├─ 同品类不同材质（石头替代豆荚→天然感保留）
  │      └─ 全新品类（陶瓷/编织→内容模板兼容性检查）
  └─ 否 → 找可用的渠道→发货测试
```

## 豆荚案例

| 步骤 | 结果 |
|:--|:--|
| 市场验证 | ✅ 8.8%互动率 |
| TK合规 | ✅ 不违禁 |
| 云途 | ❌ 植物制品禁运 |
| DHL | ❌ 同样禁运 |
| 燕文 | ❌ 同样禁运 |
| 竞争判断 | 护城河→护城河变成死路→换品 |
| 换品方向 | 抛光度小石头/水晶·保留天然感+视频模板兼容 |

## 原则

- **路堵了等于信息来了。** 堵得越快，省的时间越多
- **不坐以待毙。** 等物流政策变化=被动。立刻换品=主动
- **内容模板是投资。** 验证过的视频模板不因换品而废——找能兼容的产品
