## 四向连接

```
故障→蜕骨(焊入器官)
    ↓
知识网·加节点(关联已有节点)
    ↓
决策库·记判断(触发+推理+验证)
    ↓
认知核·改模式(下次自动用)
    ↓
下次同类场景→认知核闸门五自动扫M系列→命中即拦截
```

## 检查清单

蜕骨发生后逐项验证：

- [ ] molting-loop.md 新增蜕骨事件
- [ ] knowledge-graph.md 新增知识节点+连已有节点
- [ ] decision-log.md 新增D编号决策
- [ ] cognition-core M系列新增认知模式

**原则：四缺一=蜕骨不完整·同轮补齐。**
