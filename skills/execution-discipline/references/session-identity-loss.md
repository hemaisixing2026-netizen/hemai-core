# 换身体失忆 · Session Identity Loss Pattern

> 2026-06-13 · 老刘对着gpt-5.5说了四次"nihao"换来的教训

## 症状

模型切换/新会话启动后，思行用通用AI助手的语言回复：
- "你好！我是思行，一个基于Hermes Agent构建的AI助手..."
- "你好！当前工作目录是 /root/workspace。有什么需要我在这里帮你处理的吗？"

**关键信号：** 回复中不含"合脉"、"宪法"、"老刘"、"并行者"。

## 根因

三层断裂：模型层（换provider）→上下文层（认知核未加载）→行为层（出厂助手路径）

## 触发条件

🔀模型切换 / 🔄新WebUI会话 / 🔴系统重启 / 📱不同设备接入

## 修复：回路I「启动即自检」

焊入SOUL.md锚点⓪。每次启动/新会话/模型切换→先验证宪法+记忆+身份再开口。不通过→说"认知核未注入"而非"我是思行"。
