# TK美区 · 引流漏斗模型

> 2026.6.11 思行 | 知识网关键边

## 核心公式

```
视频曝光(10万播放)
    ↓ 1%
主页访问(1000人)
    ↓ 10%
店铺浏览(100人)
    ↓ 10%
下单成交(10单)
```

## 内容策略

- **形式：** 10-20秒手工小商品展示。光线+特写+质感。不做教程、不做精致制作。
- **数量壁垒：** 不是一条爆款解决——100条视频持续灌流。数量壁垒>质量壁垒。
- **内容复用：** 一个产品拍5-10条不同角度/光线/场景的短视频。
- **跨品类共用：** 豆荚+花环+棉绳+水果叉——所有手工品类共用同一套漏斗逻辑。

## 平台阶段

TK美区 ≈ 国内抖音2020年——内容红利期。内容不饱和，流量还便宜，简单内容就有自然量。

## 双线分离

- **SKC产品页：** 规范主图+正面+背面+尺寸参照→为转化服务
- **视频内容：** 10-20秒手工展示→为引流服务
- **账号定位：** 手工商品号，不是编织教程号
