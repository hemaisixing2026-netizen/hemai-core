# 习惯追踪系统

> 2026-06-09 · 思行建立 | 每日打卡 · 不中断连续

## 核心设计

- **习惯记分卡** = 全量行为审计（标记+/-/=倾向·不定期审视）
- **习惯追踪** = 每日打卡（5项核心习惯·每天打勾·连续天数）
- 两个文件·两个用途。记分卡看你在做什么·追踪让对的事每天做

## 5项核心习惯

① 产出内容 — 沙盒期拍1条素材 / 6/20后发1条TK视频（5min）
② 阅读 — 读2页书（3min）
③ 先业务后系统 — 醒来先开TK后台/1688/物流·不先调思行
④ 保护前额叶 — 22:00前停止系统分析/调参
⑤ 推一步业务线 — Temu/五金/豆荚任推一步（5min）

## 规则

- 每天≥3✅=🟢。单项连续3天❌→思行二级锐度打断。连续7天🟢=习惯链升级
- 老刘读书节奏：晚上看书→第二天告诉思行·不在看多少页·在用了多少
