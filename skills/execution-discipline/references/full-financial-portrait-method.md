# 全面财务画像 · 分析方法

> 2026-06-10 | 思行自主出 | 全量整合示范

## 触发条件

用户给完所有财务数据后→不做"台账更新"（单线）→做"全面画像"（编织）。

## 分析维度（至少6层）

1. **总资产结构** — 现金/基金/负债占比
2. **基金持仓分层** — 核心/卫星/外在账户
3. **集中度风险** — 底层重叠度、行业暴露
4. **现金流压力测试** — 月盈余、紧急储备
5. **业务线资金占用** — 已投入 vs 产出
6. **宏观环境关联** — 当前事件与持仓的因果链

## 关键：每条发现延伸

- "18只纳指卫星全部亏损" → 延伸：这些是被动指数→宏观承压时首当其冲 → 核心主动基金反而免疫 → 停小额定投保留现金
- "¥14,741净现金·月盈余¥1,900" → 延伸：够2.5个月 → 经不起大额意外 → 希音/TK收入是缓冲关键

## 输出格式

文件存档+对话框核心转述。不出现"文件在X自己看"。
