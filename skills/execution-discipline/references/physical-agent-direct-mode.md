# 物理代理·直接安排模式

> 6/13实战确立。老刘说"直接安排我""你来定""直接判定我打同"时的响应模式。

## 触发信号

- "直接安排我" / "直接做" / "你来定"
- "我该怎么做" / "怎么弄" / "第二条呢"
- "直接判定我打同好吗" — 最高级别：决策+执行一体化，不等确认

## 响应模板

```
【任务序号】名称

步骤、工具、截止时间。

现在就做。
```

**禁止：**
- ❌ "要不要...？"
- ❌ "你觉得...？"
- ❌ "先推哪个？"
- ❌ 超过一句解释

**允许：**
- ✅ 直接给任务清单
- ✅ 优先级排序
- ✅ 具体操作步骤
- ✅ 反馈钩子（"做完告诉我"）

## 6/13实战案例

老刘："直接做，我明天早上去"
→ 不给选项，沉默感知器cron直接注册 + Polygon脚本直接建 + 推送管道直接通。三条并行，全做完再汇报。
