# TK美区 FBT 操作指南

> 2026-06-08 | FBT = Fulfilled by TikTok（类似亚马逊FBA、Temu全托管）
> 来源：seller-us.tiktok.com, hyperfocus.tech, cahoot.ai, productlair.com, printify.com

---

## 一、FBT 是什么

货送到TK官方仓 → 平台负责：仓储、拣货、打包、发货、退货处理。你只管：选品、备货、定价、内容、投流。

## 二、费用

| 费用项 | 标准 | 说明 |
|:--|:--|:--|
| **履约费** | **$3.58/单起** | 最轻量级，按重量递增 |
| 多件订单 | $2.86/件起 | 同一买家多件 |
| **仓储费** | **前60天免费** | 60天后按体积/时长收费 |
| **运费** | **免费** | FBT订单买家免运费（2025.6.16起） |
| 退货处理费 | 按件计 | 2026.6.1起改为按件计费 |
| 重量上限 | 50 lb (22.7kg) | 单件不超过 |
| 尺寸上限 | 26 in (66cm) | 最长边不超过 |
| 按实重计费 | ≤2 lb 且 ≤332 in³ | 超出的按体积重 |

*来源：hyperfocus.tech (2026.3.20), seller-us.tiktok.com*

## 三、入驻 & 操作流程

### Step 1: 开通FBT
- TK Seller Center → 物流设置 → Fulfilled by TikTok
- 完成注册（简单注册流程，来源：seller-us.tiktok.com）

### Step 2: 创建入库计划
- 选择产品SKU → 创建FBT入库单
- 系统分配目标仓库
- 打印FBT标签（每个产品贴标）

### Step 3: 备货发货
- 按入库计划打包
- 发往TK指定仓（美国境内仓库）
- 建议：每款50-100件起步测试（来源：productlair.com）

### Step 4: 入库后
- TK仓收货扫描 → 库存上架
- 出单后平台自动发货
- 退货回TK仓处理

## 四、FBT vs Temu全托管 对比

| | Temu全托管 | TK FBT |
|:--|:--|:--|
| 定价权 | ❌ 平台定 | ✅ 你自己定 |
| 内容/流量 | ❌ 平台分配 | ✅ 你自己做视频+投流 |
| 品牌露出 | ❌ 无 | ✅ 在你的TikTok账号下 |
| 备货逻辑 | 1688采购→入仓 | **一模一样** |
| 履约费 | 平台扣点 | $3.58/单起 |
| 仓储 | 平台仓 | TK仓（前60天免） |
| 退货 | 平台处理 | 退回TK仓 |

## 五、对SquishyBean的实操建议

1. **第一批货**：选之前Temu上跑过数据的款式，50件起测
2. **定价**：1688采购价×2.5-3倍（覆盖$3.58履约费+平台佣金+利润）
3. **内容**：ASMR+天然材料视频已经在做，直接复用TK号
4. **入库**：国内→美国仓（走货代/国际物流），到仓后TK接管
5. **节奏**：6/20恢复发视频 → 同步备第一批FBT货 → 开卖
