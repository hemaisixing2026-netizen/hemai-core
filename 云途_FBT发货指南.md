## 二、FBT包装要求 ⚠️ 严格

| 要求 | 说明 |
|:--|:--|
| **独立包装** | 每件商品独立包装，不能裸奔。豆荚挂件用气泡袋/珍珠棉 |
| **外箱** | 用瓦楞纸箱，不能破损、变形。推荐邮政3号箱或定制小箱 |
| **填充物** | 箱内空隙用气泡膜、泡沫、纸团填满，晃动无声 |
| **封箱** | 透明胶带封口，不能用彩色胶带或编织带 |
| **标签** | FBT标签贴在箱子最大面，四角对齐，二维码清晰 |
| **禁用品** | 不能用订书钉、橡皮筋、稻草填充、塑料绑带 |
| **FNSKU** | 每个商品贴FBT商品条码（TK后台生成） |

**红线**：标签不清晰→拒收。包装破损→拒收。裸奔→拒收。退货运费你出。
