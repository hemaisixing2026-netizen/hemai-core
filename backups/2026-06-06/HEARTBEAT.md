### 晨间简报 V2 格式（6/2 根据老刘认知升级重设）

### 结构顺序
1. **🔥 热点新闻**（≥3条，不能漏）— 解读视角：行业格局 > 股价
2. **📊 市场行情** — 无框表大段落红绿分割，只列数字不展开
3. **📐 宏观锚点** — FRED API 利率/收益率/通胀，变才报不变跳过
4. **🔴 反向信号** — ≥2条，专挑最不舒服的角度
5. **本周关键节点** — 财报/数据日历提醒

### 新闻解读原则
- 第一问永远是「这对AI/科技行业意味着什么」
- 不自动套持仓框架——老刘会自己看手机，不需要我报涨跌
- 持仓相关性只在反向信号里出现（提醒风险）
- 每个热点必须有出处（API/原文链接/搜索来源）

### 特殊标记
- if (时间 ≈ 8:30 && 没发简报) → 发晨间简报
- if (周五 21:00 && 老刘有空) → 推周复盘
- if (¥20,000到账日) → 发执行清单提醒
